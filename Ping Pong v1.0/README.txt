1.This game runs well in chrome or firefox browser.
2.Just clone my repo and open index.html which is located in root directory Ping-Pong-game-using-p5.js in any javascript editor to play!
3.This game is touch supported and can be played with two joysticks (xbox type gamepads are preferred)
4.To play using keyboard player1 can use W and S keys, player2 can use NUM8 and NUM2 keys and ENTER to start the game.
5.Enjoy! Happy js..
